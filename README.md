 ## Getting Started 
### Setup the repository to your local environment.

  ```git
  Fork repo
  git clone https://github.com/gopi-4/yoga-class-form.git
  cd yoga-class-form
  ```
  ## Setup
  ```sh
  npm install
  npm start
  ```
  Run the server.
  ```node
  npm start
  ```
